1.Bubble Hit Game
A fun and interactive browser-based bubble game built using HTML, CSS, and Vanilla JavaScript.

2.Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS, DOM Manipulation)

3.How to Play
You are given a target number (Hit Number).
Multiple bubbles with random numbers (0â€“9) appear on the screen.
